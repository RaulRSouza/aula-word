# Aula de Word — FUNDAT Informática Básica 2026

Atividade interativa para a aula de Microsoft Word (Seções 5.0 a 5.14).

## 🔗 Acesse a aula

**[Clique aqui para abrir a aula](https://SEU-USUARIO.github.io/SEU-REPOSITORIO/)**

## 📋 Conteúdo

- 5.0 Microsoft 365
- 5.1 O Word (história, formatos de arquivo)
- 5.2–5.4 Interface e Componentes da Janela
- 5.5 Guia Arquivo
- 5.6 Guia Página Inicial (Fonte, Parágrafo, Estilos)
- 5.7 Guia Inserir (Tabelas, Imagens, SmartArt)
- 5.8 Guia Layout (Margens, Orientação, Colunas)
- 5.9 Guia Referências (Sumário, Notas, Citações)
- 5.10 Guia Correspondências
- 5.11 Guia Revisão (Ortografia, Controlar Alterações)
- 5.12 Guia Exibir (Modos de Exibição, Macros)
- 5.13 Guias Extras (Contextuais)
- 5.14 Teclas de Atalho

## 🚀 Como hospedar no GitHub Pages

1. Crie um repositório no GitHub (ex: `aula-word`)
2. Faça upload do arquivo `index.html`
3. Vá em **Settings → Pages → Source: main branch / root**
4. Aguarde ~1 minuto e acesse o link gerado

## ⏱ Duração

4 horas — com teoria, atividades práticas e quiz por módulo.
